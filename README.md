# Pocket Planter

A dynamic marketing and e-commerce website for **Pocket Planter** — a fun, pocket-sized seed planting tool for kids aged 5–16.

**From Pocket — To Plant.**

## Features

- Product storytelling and "how it works" walkthrough
- Interactive image gallery with lightbox
- Shopping cart with add-on bundles (customised seeds, fertilised water)
- Checkout flow with order confirmation
- Fully responsive, mobile-first design

## Live Site

Published via GitHub Pages.

## Local Development

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
